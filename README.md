## Start

1. Clone
```sh
   git clone https://github.com/Maryna-Serhiichuk/digi-core.git
```

2. Go to folder
```sh
   cd frontend
```

3. Start dev mode
```sh
   yarn dev
```

4. Open
```sh
   http://localhost:3000
```
